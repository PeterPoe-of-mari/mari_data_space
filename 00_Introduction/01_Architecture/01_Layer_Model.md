# Layer Model

1. **Value Store** – raw, atomic values (append-only; no semantics, origin, or norms)
2. **Mari Index** – machine-native memory (append-only; primitives/markers; non-normative), für more information look up mari index language repository
3. **Code of Law** – explicit, project-specific rule system (norms, rights, obligations)
4. **Order Layer** – optional order metadata (origin/structure; human-readable; coexisting)
5. **Projections** – context- and role-dependent views/responses (derivable; ephemeral)
