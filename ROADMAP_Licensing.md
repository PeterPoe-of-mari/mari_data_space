# Roadmap: Licensing of the Overall Project

Goal: **Standards-friendly specification**, **Commons-protected reference implementation**, **open datasets**.

## Target Structure (Repositories and Licenses)

1. `mari-index-language` → CC BY 4.0  
2. `mari-data-space` → CC BY 4.0  
3. `mari-engine` → GPLv3  
4. `mari-datasets` → ODbL  

## Milestones

- M1: Finalize CC BY 4.0 licensing in `mari-index-language` and `mari-data-space`
- M2: Initialize `mari-engine` (GPLv3) + Minimal CLI
- M3: Initialize `mari-datasets` (ODbL) + ERP Demo v0.1
- M4: Harmonize Contribution Guidelines / Authors / Changelog
